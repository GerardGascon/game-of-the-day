Frequently Asked Questions

  • How can I contribute? add
    You can contribute in two different ways, first one is by submitting a game via this form. And also, you can make a small donation at my Ko-fi page in order to help me to pay the small costs this project needs.
    Can I submit a game that is not mine? add
    Yes, you are free to submit any game you want. If you do so, please tell me in the form why you think that game should be in the website.
    What happens if I submit more than 3 games? add
    I don't strictly follow the rule of a 3-game limit, but if I see there are too many games from you, I will send an email asking you which game you would like to remove before accepting the new entry.
    What happens if another person has already submitted 3 of my games? add
    I will always prioritize the developer's choice. So if you, as a developer, want to add a game but another person has already submitted 3 of yours, I will contact you telling the situation and ask which games to keep.
    Does the game's description have to be en English? add
    Not necessarily, as I said in the About page, your game can be in any language you want. In this page you will find games in English, French or even Catalan!
