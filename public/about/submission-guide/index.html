Submission Guide

To submit your game to the website you first have to go to the Issues Page and submit a new issue following the Game Submission Template.

Write your game's title and link to the download page, remember that your link must have the https:// prefix in order to work properly.

Then add the store page's name where it is available. It can be itch.io, Steam, Game Jolt, or anything else, this is used for telling where it's hosted inside the website.

Drag and drop your cover image inside the Cover Image field. DO NOT write anything else in there or the issue might not get parsed correctly.

Write a short description of your game, it should not surpass the 120-character limit.

This limitation is not strictly applied but if your description is way beyond that limit, it may be rejected.

After that, add your screenshots the same way you added your cover image, remember that your game can have up to 3 screenshots, if you have more than 3, your submission will be rejected.

Finally, you'll have to add the developers that participated in the development of the game, as well as their social network link (or whatever link that references them).

Important: Please pay attention how this field is written.

If the developer has socials, you should write their name as so: Name - Link

If they don't have socials, you should write just their name as so: Name

Note: Even though the screenshots don't show it, the developers' names can have spaces if you need to.

Revision

After you submit your game, an action will be automatically triggered and if everything is correct I'll receive an email telling me to review your submission.

And if everything is correct, I'll add your game to the website, and it will be ready to be showcased on a random game that nobody knows!
